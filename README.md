# Pixelify Sans for ShiggyCord

Pixelify Sans font pack for ShiggyCord.

## Install

Open **ShiggyCord → Fonts → Import Font** and paste:

https://raw.githubusercontent.com/Yubu77/pixelify-sans-shiggy/main/pixelify.json

## Repository

https://github.com/Yubu77/pixelify-sans-shiggy

## Included Fonts

- PixelifySans-Regular.ttf
- PixelifySans-Medium.ttf
- PixelifySans-SemiBold.ttf
- PixelifySans-Bold.ttf
